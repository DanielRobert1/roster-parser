Get Roster Events

This allows roster_events to view the paginated data of roster_events

Example request:
curl --request GET \
    --get "http://localhost/api/roster" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"type\": \"UNK\",
    \"arrival_location\": \"et\",
    \"destination_location\": \"labore\",
    \"metrics\": \"custom\",
    \"from_date\": \"04\\/19\\/2024\",
    \"to_date\": \"04\\/19\\/2024\",
    \"per_page\": 19
}"

const url = new URL(
    "http://localhost/api/roster"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "type": "UNK",
    "arrival_location": "et",
    "destination_location": "labore",
    "metrics": "custom",
    "from_date": "04\/19\/2024",
    "to_date": "04\/19\/2024",
    "per_page": 19
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Headers

Example: application/json

Body Parameters

type   string  optional  

Example: UNK

Must be one of:
  • FLT
  • SBY
  • OFF
  • UNK
arrival_location   string  optional  

Example: et

destination_location   string  optional  

Example: labore

metrics   string  optional  

Example: custom

Must be one of:
  • today
  • week
  • nextWeek
  • month
  • 30days
  • 90days
  • year
  • allyear
  • custom
from_date   string  optional  

Must be a valid date in the format m/d/Y. Example: 04/19/2024

to_date   string  optional  

Must be a valid date in the format m/d/Y. Example: 04/19/2024

per_page   integer  optional  

Example: 19

Upload Roster Events

This allows upload of roster to be parsed

Example request:
curl --request POST \
    "http://localhost/api/roster/upload" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"roster\": \"et\"
}"

const url = new URL(
    "http://localhost/api/roster/upload"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "roster": "et"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
